Some quick tips on inputting stuff:
  • To add an individual event (a single point), you only have to fill in "Starting Point".
  • To add a ray, you provide a starting point and speed (as a fraction of c). Leave "Ending Point" blank.
  • To add a segment, you provide a starting point and ending point. Leave "Speed" blank.
  • Color is optional, and if you don't provide one, it will default to black. However, I think colorful graphs are more fun to look at. :)
